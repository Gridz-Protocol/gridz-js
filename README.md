# gridz-js

TypeScript packages for [Gridz](https://github.com/Gridz-Protocol/gridz) — the
`@gridz/*` scope: `core`, `sdk`, `cli`, `server`, `sinks`, `mcp`, `react`, `vue`,
`svelte`, `element`, `oneclaw`.

A submodule of the [`gridz`](https://github.com/Gridz-Protocol/gridz) superproject.
Build and test from there (`pnpm -w run check`) — packages share the superproject's
`specs/`, fixtures, and base tsconfig.
